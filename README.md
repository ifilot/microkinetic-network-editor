Microkinetic Network Editor (MNE) is a desktop application for facile
visualization of microkinetic networks.

MNE helps researchers and students inspect reaction pathways by turning
microkinetic input data into a clear, interactive network diagram. The program
is designed to make it easier to:

- visualize species and reaction steps in a single network view;
- inspect connectivity and identify key pathways at a glance;
- communicate model structure through publication-ready screenshots.

To get started quickly, download the installer for your platform from the
repository's
[Releases](https://github.com/ifilot/microkinetic-network-editor/releases) tab
on GitHub.
